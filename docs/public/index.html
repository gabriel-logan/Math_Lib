Biblioteca pessoal criada em JavaScript e Python

para uso comum. Em desenvolvimento...

JavaScript

Para usar como pacote NPM, execute o seguinte comando:

npm i loganmatic

E para importar, utilize:

import Matematica from "loganmatic.js"

Ou

const Matematica = require("loganmatic.js");

Exemplo de uso:

console.log(Matematica.NumeroEuler);
console.log(Matematica.fatorar(100));

Retorno:

2.7182818284590455
[ 2, 2, 5, 5 ]

Python

Para usar como pacote PIP, execute o seguinte comando:

pip install loganmatic

E para importar, utilize:

from loganmatic import Calculadora

Exemplo de uso:

Matematica = Calculadora()
print(Matematica.NumeroEuler)
print(Matematica.fatorar(100))

Retorno:

2.7182818284590455
[2, 2, 5, 5]

Funcionalidades

A biblioteca calcula (atualmente) o valor de PI e do Número de Euler. Além disso, realiza as seguintes operações:

  • Calcula funções: Seno, Cosseno
  • Fatora um número
  • Calcula o fatorial de um número
  • Calcula o módulo
  • Gera um número aleatório entre 2 números
  • Calcula raízes cúbicas e quadradas
  • Calcula as raízes de equações de primeiro e segundo grau

Features

The library calculates (currently) the value of PI and Euler's number. It also performs the following operations:

  • Calculates functions: Sine, Cosine
  • Factors a number
  • Calculates the factorial of a number
  • Calculates the modulus
  • Generates a random number between two numbers
  • Calculates cubic and square roots
  • Calculates the roots of first and second-degree equations
